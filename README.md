# Проект: Лендинг «Мир мёда»
## Описание
Это адаптивный одностраничный сайт о пчеловодстве и продуктах из меда. В этом проекте я показала свои навыки верстки сложного текстового контента и работы с сетками.
## Что я сделала (мои навыки):
Чистый код: Верстала по правилам (теги header, section, footer), чтобы сайт лучше находили поисковики.
Методология БЭМ: Названия классов в коде логичны и понятны другим разработчикам.
Адаптивность: Сайт отлично смотрится и на компьютере, и на телефоне.
Точность (Pixel Perfect): Сайт на 100% соответствует дизайну из Figma.
## Технологии:
HTML5
CSS3 (использовала Flexbox для выравнивания блоков)
Адаптивная верстка
